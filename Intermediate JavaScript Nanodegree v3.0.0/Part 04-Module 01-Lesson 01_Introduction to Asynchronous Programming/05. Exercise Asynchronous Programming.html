05. Exercise: Asynchronous Programming

Asynchronous JavaScript Practice.

This is a quick peek ahead into some of the syntax we'll be going over in the next lesson.

Challenge 1

Move the setTimeout after the console log, does that change the outcome?

Challenge 2

Edit the time elapsed between "Hello" and "World" appearing on the screen

Challenge 3

Edit the message being sent to have three parts:

  • Part 1: "I'm " - shows up immediately
  • Part 2: "async" - shows up 2 seconds after "I'm"
  • Part 3: "…hronous!" - shows up 4 seconds after "async"

Extra Challenge

Write a function to make each dot appear one second after the other.

Reminder

To run a JavaScript file in the Workspaces use node

node asynchronous_hello_world.js
