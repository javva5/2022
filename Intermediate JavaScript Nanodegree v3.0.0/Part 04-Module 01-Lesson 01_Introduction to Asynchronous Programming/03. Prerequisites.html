03. Prerequisites

Prerequisites

To be ready for this course, you need knowledge of the following topics:

  • General beginner level JavaScript
  • General ES6 syntax
  • Understanding of callbacks
  • Basic understanding of JavaScript event handling

Resources

Here are some resources that will put you in a good place for starting this course:

  • If you need to brush up on your JavaScript skills you can take this free Udacity course: Intro to JavaScript
  • For a refresher on callbacks, here's a good article: Async JavaScript: From Callbacks, to Promises, to Async/Await.
  • For a refresher on JavaScript events, AJAX requests, and the Browser API - read chapters 13 - 15 of Eloquent JavaScript.
  • To get a glimpse of where we're going, take a look at MDN's Introduction to Asynchronous Programming.
