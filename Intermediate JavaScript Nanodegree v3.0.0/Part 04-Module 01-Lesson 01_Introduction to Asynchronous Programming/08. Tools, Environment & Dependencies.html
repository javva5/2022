08. Tools, Environment & Dependencies

Tools, Environment & Dependencies

This course focuses 100% on vanilla JavaScript, so there aren't many dependencies! Below you will find some options that might come in handy for running the code you see here in the course.

#1 - You can run code in the Udacity workspaces.
#2 - You can run code locally on your machine.
#3 - You can run code in an online code editor like ES6Console.

Running Code Locally

For these examples it will be helpful to have a fairly up to date version of Node. To see the latest stable version, go to NodeJS.org/downloads

To check if you have Node installed and see what version you are running, open a terminal and run 

node --version

Or, if you are using the node version manager NVM, run 

nvm current
