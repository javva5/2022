09. Project: UdaciRacer Simulator

UdaciRacer Project

Lets take a look at a finished example of the project and what you will be building.

ND032 JSND L0 A05 Project UdaciRacer Simulator V2

Project Walkthrough

The project for this course is to complete an interactive racing simulation game in which a user will select options for their race course and racer. The user has control over the acceleration of their racer by clicking a button, pretty much the name of the game is madly click the button as fast as you can and you should see your racer rise to the top of the leader board. Completing the functionality will involve all the skills you learn in this course, but the result will be a fun game that you can actually play and show your friends.
