07. Quizzes: Promises I

What's wrong with this?

Look at this promise chain -- what is wrong?

new Promise((resolve, reject) => {
    resolve([255, 254, 256, 257]);
})
.then((data) => {
    data.push(253)
    return data;
})
.then((data) => {
    data.push(252)
    return data
})
.then((data) => {
    console.log(data)
})
SOLUTION: Missing a `catch` clause

Promise Chaining

Which of the following are true about Promise chaining?
SOLUTION:
  • You can chain as many `then` statements as you want
  • You can add `catch` statements between any `then` statements
  • The `finally` statement runs after all other statements have finished

Promise Chaining

What would this code output?

new Promise((resolve, reject) => {
  resolve("This is a new message");
})
.then(data => {
  const dataArr = data.split("");
  throw new Error("Uh-oh!");
  return dataArr.reverse();
})
.then(data => data.join(""))
.then(data => console.log(data))
.catch(error => {
    console.log("There was an error");
    console.log(error.message);
});
SOLUTION: There was an error, Uh-oh!
