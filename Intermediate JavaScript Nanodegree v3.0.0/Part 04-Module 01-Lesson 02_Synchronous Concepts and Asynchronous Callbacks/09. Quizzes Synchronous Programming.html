Blocking or Non-Blocking?

Look at this code and determine whether it blocks the thread or not. Remember, you can run it locally to find out!

function runTimer = (startingTime) => {
  const time = parseInt(startingTime);
  while (time < 100) {
    console.log("testing", time);
    time = time++;
  }
};

runTimer(0)
console.log("timer complete");
SOLUTION: Blocking

Blocking or Non-Blocking?

Look at this code and determine whether it blocks the thread or not. Remember, you can run it locally to find out!

let recentAccountBalances = [];

const appendBalance = () => {
  recentAccountBalances.push(Math.random() * (200 - 100) + 100)
  console.log(recentAccountBalances)
}

const pollAccount = () => {
  setInterval(appendBalance, 4000)
}

console.log("test")

pollAccount()
SOLUTION: Non-Blocking
